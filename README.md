# MainCrafts Internship - MERN Stack Task 1

## Project Overview
This project is a MERN Stack CRUD application developed as part of the MainCrafts Internship Task 1.

## Features
- Add Product
- View Products
- Update Product
- Delete Product
- MongoDB Atlas Integration
- Responsive React UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Project Structure

client/
server/

## Installation

### Client

```bash
cd client
npm install
npm start
```

### Server

```bash
cd server
npm install
npm start
```
